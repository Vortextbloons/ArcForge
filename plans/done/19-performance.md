# 25. Performance Requirements

The engine should support:

```txt
Instanced meshes
Static batching
Material reuse
Texture compression
GLB/glTF asset flow
LOD groups
Frustum culling
Object pooling
Fixed-step physics
Asset preloading
Lazy loading
Runtime profiler
Editor performance warnings
```

The editor should warn about:

```txt
Too many draw calls
Too many unique materials
Large uncompressed textures
Huge scene JSON files
Too many dynamic lights
Too many physics bodies
Missing LODs
Oversized scripts
```
